# Caldrop

Event scheduling and sharing tool. Create events, generate shareable links, and handle timezone conversions automatically.

## Stack

- Next.js, TypeScript
- Supabase (PostgreSQL, Auth)
- Tailwind CSS, shadcn/ui
- Vercel

## Setup

```bash
npm install
cp .env.example .env.local  # add Supabase credentials
npm run dev
```
